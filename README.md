En el este archivo de documenta el trabajo de ejecución del programa 

Se tienen 8 Funciones que sirven de la siguiente manera
1. see_book éste sirve para mostrar los libros que están en la lista
2. add_book éste sirve para agregar nuevos libros en la lista
3. update-book con ésta función podemos actualizar los datos del libro su estan mal ingresados
4. delate_book sirve para eliminar los libros de la lista 
5. regist_sale Con éste se puede registar compras de los libros
6. top_3 Sirve para ver la cantidad de libros mas vendidos 
7. sales_by_author Con este vemos cual es el autor que mas vende libros
8. income_report Validar los datos y nos resalta los valores netos y brutos del programa
